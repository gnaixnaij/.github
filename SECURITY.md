# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any of my projects, please report it privately.

**Do not disclose it publicly until it has been addressed.**

### How to report

1. Open a draft security advisory on the affected repository:
   - Go to the repo → **Security** tab → **Advisories** → **New draft advisory**
2. Or email: gnaixnaij@users.noreply.github.com

I will acknowledge receipt within 48 hours and work on a fix before disclosure.

## Scope

All repositories under [github.com/gnaixnaij](https://github.com/gnaixnaij) are covered by this policy.

## Supported Versions

Only the latest release of each project receives security patches.
